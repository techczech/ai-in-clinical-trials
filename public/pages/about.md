# About this Project

This website was created by **Dominik Lukeš** ([dominiklukes.net](https://dominiklukes.net)) as a practical illustration of a research process powered by agentic AI. 

## Purpose
The project serves a dual purpose:
1. **Research Presentation**: Showcasing findings on the possible uses of generative AI in clinical trials.
2. **Process Illustration**: Demonstrating the methodology and tools used to gather, analyze, and present research data in the year 2026.

## Technology Stack
The site is built and maintained using modern AI and web technologies:
- **AI Agent**: Built using **Antigravity** (Google DeepMind) with the **Gemini 3 Flash** model.
- **Content Management**: Content is stored in easily maintainable Markdown files, allowing for rapid iteration and updates.
- **Source Control**: Managed via a public repository on [GitHub](https://github.com/dominiklukes/ai-in-clinical-trials).
- **Hosting**: Deployed and served via **Cloudflare Pages**.

## Accountability
A detailed log of the research process, including specific prompts, tool choices, and agent interactions, is preserved in the **[Process Notes](https://github.com/dominiklukes/ai-in-clinical-trials/blob/main/processnotes/ProcessNotes.md)**.
